# 🧠 AlphaPilot X

AlphaPilot X is an AI-powered trading assistant project designed to help with market analysis, risk management, strategy planning, and trading education.

## 🚀 Features

- Risk calculator
- Trading strategy assistant
- Memory system
- Market analysis modules
- Trade planning tools
- Modular Python architecture

## 📂 Project Files

- `alphapilot.py` - Main program
- `brain.py` - AI logic
- `memory.py` - Memory system
- `strategy.py` - Trading strategies
- `market.py` - Market tools
- `trading.py` - Trading module
